Generation Methodology
🔬
Persona-conditioned probabilistic sampling. 330 per-persona distributions extracted from 2,000 original rows, used as generation priors with Dirichlet noise.
Step 1 — Extract Per-Persona Distributions
All 330 per-persona categorical and binary distributions extracted from the 2,000-row original. Dirichlet priors formed from real frequencies.
Step 2 — Dirichlet Noise Injection
Dirichlet noise (α = real_freq × 80 + 0.5) added to each categorical prior to simulate natural survey variance.
Step 3 — Persona-Conditioned Sampling
Each of the 10,000 rows first assigned to a persona, then all 55 columns sampled from that persona's distributions independently.
Step 4 — Missing Value Replication
style_secondary ~51.4% missing and four binary columns ~0.9% missing — replicated exactly from original patterns.
Step 5 — Statistical Validation
Persona proportions exact (±0%), binary column means within ±3.5%, sustainability score ordering preserved, missing rates matched.

Model Evaluation Protocol
80/20 Train/Test Split
Stratified by persona and purchase_intent for balanced class representation.
5-Fold Cross-Validation
Reported metrics are mean CV scores to reduce variance from random splits.
SHAP Explainability
TreeExplainer applied to XGBoost to compute Shapley values for each feature.
CLV Estimation
12-month CLV = Avg Order Value × Purchase Frequency × (1 − Churn Rate). Segmented by persona.
